# Portfolio Flutter Scaffold

Responsive Flutter web starter for a portfolio homepage.

## Included
- Widget-first architecture
- Centralized theme tokens
- `go_router` web routing setup
- Desktop / small desktop / tablet / mobile breakpoints
- Dynamic headline hero
- Terminal-inspired about section
- Project card section with dummy visuals

## Next steps
1. Run `flutter pub get`
2. Run `flutter run -d chrome`
3. Replace dummy project data with real case studies
4. Add routes for content pages with `go_router`
5. Add SEO enhancements such as metadata handling during deployment
